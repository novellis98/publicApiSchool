# Spring Boot Training Project

## Descrizione

Questo progetto è una API REST sviluppata con Spring Boot con l’obiettivo di migliorare le mie competenze backend.

Non è un semplice CRUD: lo sto usando per esercitarmi su aspetti reali dello sviluppo, come progettazione delle query, performance e gestione delle relazioni.

---

## Obiettivi

Sto lavorando su:

* Criteria API per query dinamiche
* Ottimizzazione delle query (evitare N+1)
* Gestione relazioni JPA
* Uso corretto dei DTO
* Paginazione, sorting e filtri
* Entità intermedie (Enrollment con chiavi composte)

---

## API pubbliche (in sviluppo)

L’obiettivo è rendere disponibili alcuni endpoint pubblici per permettere a chi sviluppa frontend di esercitarsi con:

* fetch / axios
* integrazione con un backend reale
* gestione dati lato client

---

## Cosa sto imparando

Questo progetto mi serve per:

* capire cosa genera Hibernate a livello SQL
* scrivere query efficienti e scalabili
* evitare problemi comuni (come N+1)
* ragionare da backend developer, non solo “far funzionare il codice”

---

## CI/CD

Sto approfittando di questo progetto anche per iniziare a lavorare su:

* pipeline CI/CD
* build automatizzate
* possibili deploy futuri

---

## Stack

* Java
